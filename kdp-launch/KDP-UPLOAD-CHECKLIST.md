## 🔧 COMMON ISSUES & FIXES

### **Issue: Upload fails**
**Fix:** 
- Check file size (<50 MB)
- Verify format (.docx, .epub, no .pdf)
- Remove special characters from filename
- Try different browser

### **Issue: Cover rejected**
**Fix:**
- Verify dimensions (2560x1600px minimum)
- Check file size (<50 MB)
- Ensure no prohibited content (violence, nudity)
- RGB color space (not CMYK)

### **Issue: Manuscript has errors**
**Fix:**
- Download error report from KDP
- Fix in Word/Atticus
- Re-upload
- Test in Kindle Previewer first

### **Issue: Can't select 70% royalty**
**Fix:**
- Check price is $2.99-$9.99
- Verify enrolled in KDP Select OR worldwide rights
- Confirm file size <10 MB

### **Issue: Description formatting broken**
**Fix:**
- Remove fancy formatting
- Use only: `<b>`, `<i>`, `<br>`
- Test with plain text first
- Re-paste if needed

### **BEFORE CLICKING PUBLISH:**

1. **Triple-check everything:**
   - [ ] Title spelling (can't change easily after)
   - [ ] Cover looks good at thumbnail
   - [ ] Description has no typos
   - [ ] Keywords make sense
   - [ ] Price is competitive

2. **Preview thoroughly:**
   - [ ] Use Kindle Previewer tool
   - [ ] Check on phone (most readers use mobile)
   - [ ] Test TOC links
   - [ ] Verify chapter breaks

3. **Have backup plan:**
   - [ ] Save all source files
   - [ ] Can update interior anytime
   - [ ] Can change price anytime
   - [ ] Can update description anytime

### **AFTER PUBLISHING:**

4. **Don't panic if:**
   - Sales are slow first week (normal)
   - No reviews immediately (takes time)
   - Rank is low initially (builds with sales)

5. **DO track:**
   - Which keywords drive traffic (Amazon Ads later)
   - Which categories perform best
   - KU vs purchase ratio
   - Review sentiment

6. **First week goals:**
   - Get 3-5 reviews (friends, family, beta readers)
   - Monitor ranking daily
   - Share on social media 3-4 times
   - Engage with any comments/questions
