## 🛠️ FORMATTING TOOLS

### **OPTION 1: MICROSOFT WORD (Recommended)**

**Step-by-step:**

1. **Open your manuscript** (killer-clown-FINAL.md)
2. **Copy all text** to Word document
3. **Apply styles:**
   - Font: Times New Roman 12pt
   - Line spacing: 1.5
   - Margins: 1" all sides
   - First line indent: 0.3"

4. **Add page breaks:**
   - Before each chapter: Insert → Page Break
   - Before front matter sections
   - Before back matter sections

5. **Format chapter titles:**
   - Font: Times New Roman
   - Size: 18pt
   - Bold
   - Centered or Left-aligned
   - ALL CAPS optional

6. **Create table of contents:**
   - Manual (type it out) OR
   - Automatic (Word → References → TOC)

7. **Save as .docx**

## 🚫 COMMON MISTAKES TO AVOID

**DON'T:**
- ❌ Use justified alignment (causes weird spacing on Kindle)
- ❌ Add manual page numbers (Kindle ignores them)
- ❌ Include headers/footers (won't display)
- ❌ Use fancy fonts (readers can change fonts anyway)
- ❌ Add multiple blank lines (looks bad on different devices)
- ❌ Center everything (hard to read)
- ❌ Use tabs instead of first-line indent
- ❌ Include images without optimization (slows loading)

**DO:**
- ✅ Keep formatting simple and clean
- ✅ Use consistent chapter structure
- ✅ Add page breaks between chapters
- ✅ Test on Kindle Previewer before uploading
- ✅ Use standard fonts (Times/Georgia/Bookerly)
- ✅ Left-align body text
- ✅ First-line indent paragraphs
- ✅ Include hyperlinked TOC

## 📋 QUICK WORD FORMATTING STEPS

**If doing manually in Word:**

1. **Paste manuscript** into blank Word doc
2. **Font:** Times New Roman 12pt
3. **Margins:** Layout → Margins → Normal (1" all sides)
4. **Line spacing:** Home → Line Spacing → 1.5
5. **Paragraph indent:** Layout → Paragraph → First Line: 0.3"
6. **Add page breaks:** Before each chapter (Ctrl+Enter or Cmd+Return)
7. **Format chapter titles:** Bold, 18pt, centered
8. **Add front matter:** Title page, copyright, TOC
9. **Add back matter:** Author bio, review request
10. **Save as .docx**
11. **Test in Kindle Previewer**
12. **Upload to KDP**

## 💡 PRO TIPS

5. **Update easily:** You can always update interior after launch
6. **Series consistency:** Use same formatting for Books 2-10
