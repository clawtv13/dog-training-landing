## OPTIMIZATION TIPS

**After generation:**
1. Resize to exactly 1200x630px
2. Compress to <200KB (TinyPNG.com)
3. Save as: `post-slug-featured.jpg`
4. Add to `blog/images/` folder
5. Update HTML: `<img src="/blog/images/post-slug-featured.jpg" alt="descriptive text">`

**SEO:**
- Always add descriptive alt text
- Include focus keyword in filename
- Compress for fast loading
- Use WebP format if possible (with JPG fallback)
