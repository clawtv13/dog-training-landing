# Newsletter: Crianza Consciente para Padres Separados

Basada en "El Cerebro del Niño" de Daniel J. Siegel y Tina Payne Bryson.

## Concepto

Newsletter diaria para padres separados que quieren criar hijos emocionalmente sanos a pesar de las dificultades de la co-parentalidad.

## Pilares del contenido

1. **Neurociencia práctica** - Conceptos del libro aplicados al día a día
2. **Co-parentalidad** - Estrategias para dos hogares
3. **Regulación emocional** - Para padres e hijos
4. **Comunicación** - Entre ex-parejas y con los niños

## Estructura de cada email

- **Asunto llamativo** (emocional o curiosidad)
- **Historia/situación real** (2-3 párrafos)
- **Concepto del cerebro del niño** (1-2 párrafos)
- **Estrategia práctica** (acción concreta)
- **Frase para recordar** (quote o mantra)

## Frecuencia

Diaria (5 días/semana) o 3x/semana

## Tono

- Empático, sin juzgar
- Científico pero accesible
- Esperanzador
- Práctico
