## 🛠️ System Architecture

### Master Automation (`growth-engine.sh`)
Orchestrates everything:

1. **Content Generation** (`blog-auto-post.py`)
   - Selects top content from research database
   - Generates 800-1,200 word posts with Claude
   - Creates SEO-optimized HTML
   - Updates index

2. **Viral Injection** (`add-viral-mechanisms.py`)
   - Adds share buttons to all posts
   - Injects referral tracking
   - Adds social proof counters
   - Embeds CTAs

3. **Distribution** (`crosspost-platforms.py`, `smart-reddit-distribute.py`)
   - Crossposts to Medium/Dev.to
   - Posts to Reddit (smart rotation)
   - Shares on social media
   - Tracks all distributions

4. **Metrics** (`growth-tracker.py`)
   - Collects all metrics
   - Generates reports
   - Calculates projections
   - Provides recommendations

5. **Deployment**
   - Commits to git
   - Pushes to GitHub
   - Triggers GitHub Pages deploy
   - Sends Telegram notification

**Runs:** Twice daily (10am & 4pm UTC)  
**Duration:** 5-10 minutes per run  
**Manual intervention:** None required
