## 📋 Templates

### 1. ChatGPT Prompts for Business

**Email Responder:**
```
You are my email assistant. Respond to this email professionally:

[paste email]

Context about my business: [your context]

Tone: [professional/friendly/brief]
```

**Content Creator:**
```
Create a [blog post/social post/newsletter] about [topic].

Target audience: [your audience]
Word count: [800-1200]
Tone: [conversational/professional]

Include:
- Actionable takeaways
- Specific examples
- Call to action
```

**Task Prioritizer:**
```
I have these tasks today:

[list tasks]

Help me prioritize based on:
- Impact on revenue
- Time required
- Dependencies

Format as a schedule from 9am-5pm.
```

### 2. Zapier Workflows (Copy & Paste)

**Email → Task Manager:**
```
Trigger: New email in Gmail with label "Action"
Filter: From specific addresses
Action: Create task in Notion/Todoist
Map: Subject → Title, Body → Description
```

**Social Media Autopilot:**
```
Trigger: New RSS item (your blog)
Action 1: Format with ChatGPT
Action 2: Post to Twitter
Action 3: Post to LinkedIn
Action 4: Post to Facebook
Delay: Stagger 2 hours between platforms
```

**Invoice Automation:**
```
Trigger: Form submission (Google Forms)
Action 1: Calculate total (Formatter)
Action 2: Create invoice (PDF)
Action 3: Send email (Gmail)
Action 4: Save to Drive
Action 5: Add to spreadsheet
```

### 3. Notion Templates

**AI Project Tracker:**
```
Database Properties:
- Project Name (Title)
- Status (Select: Planning/In Progress/Done)
- AI Tool Used (Multi-select)
- Time Saved (Number)
- Cost (Number)
- ROI (Formula: Time Saved × Hourly Rate - Cost)
- Notes (Text)
```

**Content Calendar (AI-Powered):**
```
Database Properties:
- Content Title (Title)
- Platform (Select)
- Status (Select)
- Publish Date (Date)
- AI Prompt Used (Text)
- Performance (Number)
- Repurpose To (Relation to other content)
```

## 🚀 Quick Wins (Implement Today)

### 1. Email Automation (15 mins)
1. Create Gmail filter: "needs reply"
2. Use Zapier to trigger ChatGPT
3. Generate draft response
4. Review & send

**Time saved:** 1 hour/day

### 2. Meeting Scheduler (10 mins)
1. Set up Calendly
2. Connect to Google Calendar
3. Add to email signature
4. Let AI handle scheduling

**Time saved:** 30 mins/day

### 3. Content Repurposing (20 mins)
1. Write one long-form post
2. Use ChatGPT to create:
   - 10 tweets
   - 3 LinkedIn posts
   - 1 newsletter section
   - 5 Instagram captions

**Time saved:** 2 hours/week

### 4. Social Media Automation (30 mins)
1. Connect blog RSS to Buffer
2. Set posting schedule
3. Let it run automatically

**Time saved:** 1 hour/day

### 5. Custom AI Assistant (20 mins)
1. Go to ChatGPT → Explore GPTs
2. Create custom GPT for your business
3. Add context & instructions
4. Use for all business tasks

**Time saved:** Varies, but significant
