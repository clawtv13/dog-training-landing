## RESEARCH & ANALYSIS (8 Prompts)

### 21. Competitor Analysis Deep Dive

**Copy-Paste Prompt:**
```
Analyze these 3 competitor websites and create comparison matrix:

Competitors: [URL1], [URL2], [URL3]

Extract for each:
- Primary value proposition (1 sentence)
- Pricing model + tiers
- Top 3 features they emphasize
- Target customer (who they're selling to)
- Gaps I could exploit (what they're missing)

Format as table. Include "Opportunity" column for how I differentiate.
```

**When to Use:** Monthly or when entering new market. 15 minutes vs 2 hours manual research.

**Time Saved:** 1.75h per analysis × 4/year = 7h/year

**ROI:** 7h × $40 = $280/year

**Example Input:** Three SaaS competitor URLs in your space

**Example Output:**
| Competitor | Value Prop | Pricing | Top Features | Target | Opportunity |
|------------|------------|---------|--------------|--------|-------------|
| CompanyA | "Automate sales" | $99/mo | CRM, Email, Reports | Enterprise | They ignore solopreneurs |

**Customization:** Add "Tech stack" row if you need implementation details. Add "Content strategy" if analyzing marketing approach.

**Common Mistakes:** Analyzing too many competitors (3-5 max). Focusing on features instead of positioning.

### 25. Product Feature Prioritizer

**Copy-Paste Prompt:**
```
Prioritize these feature requests using RICE framework:

Features:
[List 10-15 feature requests]

For each, estimate:
- Reach: How many customers benefit? (scale 1-10)
- Impact: How much does it move the needle? (scale 1-3)  
- Confidence: How sure are we? (% as decimal 0.5-1.0)
- Effort: Dev time in weeks (1-12)

Calculate RICE score: (Reach × Impact × Confidence) / Effort

Return ranked by score, include reasoning for top 3.
```

**When to Use:** Sprint planning, roadmap decisions.

**Time Saved:** 1h debate → 10 min

**ROI:** 50 min × 12 sprints = 10h/year = $400/year

### 51. Website Copy Audit

**Copy-Paste Prompt:**
```
Audit my website copy for conversion optimization:

URL: [your site] or [paste homepage text]

Page type: [homepage / landing / pricing / about]

Analyze:
- Headline clarity (clear value prop in 5 seconds?)
- Above-fold content (compelling or generic?)
- Call-to-action strength (specific or vague?)
- Trust signals (testimonials, logos, guarantees?)
- Objection handling (addresses common concerns?)

For each issue found:
- Current text (quote exact)
- Problem (why it's weak)
- Rewrite (improved version)
- Why better (specific improvement)

Format as: Section | Issue | Fix | Impact
```

**When to Use:** Before major launches or when conversion rate drops.

**Time Saved:** 2h professional copywriter → 15 min self-audit  
**ROI:** 1.75h × 4 audits/year = 7h = $280/year (plus avoided $500-1000 copywriter fee)

**Example:**
```
Input: Homepage headline "We help businesses grow with AI"
Problem: Vague, no specific benefit, everyone says this
Rewrite: "Cut admin work from 20h/week to 5h using AI automation"
Why better: Specific number, clear before/after, unique positioning
```

**Customization:** Add "Mobile vs desktop" analysis if traffic split matters. Include "Above/below fold" sections if bounce rate high.

**Common Mistakes:** Auditing entire site at once (focus on highest-traffic pages first). Rewriting everything immediately (test one change at a time, measure impact).
