## 🎓 SEO Knowledge Transfer

### For Future Content Creators

**New Post Checklist:**
- [ ] Title < 60 characters
- [ ] Meta description 120-160 chars
- [ ] Target keyword in title, H1, first paragraph
- [ ] 2+ H2 headings with keywords
- [ ] 800+ words (prefer 2000+)
- [ ] 3+ internal links
- [ ] All images have alt text
- [ ] JSON-LD structured data
- [ ] Breadcrumbs visible
- [ ] Canonical URL set
- [ ] Add to sitemap.xml
- [ ] Add to rss.xml
- [ ] Add to posts/index.json
- [ ] Run `python3 scripts/seo-audit.py`
- [ ] Request Google indexing

### SEO Best Practices

1. **Write for humans first, search engines second**
2. **Target one primary keyword per post**
3. **Use keywords naturally (no stuffing)**
4. **Internal link to related posts**
5. **Update old posts with fresh data**
6. **Monitor Search Console weekly**
7. **Build quality backlinks, not quantity**
8. **Optimize for mobile always**
9. **Focus on user experience**
10. **Be patient - SEO takes 3-6 months**
