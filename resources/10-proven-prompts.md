## 1️⃣ CONTENT SCRIPT GENERATOR

```
Generate a 60-second video script about [TOPIC].

Structure:
- HOOK (0-3s): Grab attention
- PROBLEM (3-15s): Identify pain point
- SOLUTION (15-45s): Provide actionable fix
- AUTHORITY (45-53s): Cite source/proof
- CTA (53-60s): Clear next step

Tone: [Direct/Calming/Controversial]
Audience: [Demographics]

Include:
- Specific numbers/data
- Authority sources (NASA, Stanford, etc.)
- Clear visual cues for video editing
```

## 2️⃣ COST OPTIMIZATION AUDITOR

```
Analyze my OpenClaw usage from the past 30 days.

Review:
- Total tokens consumed by model
- Tasks that could use cheaper models
- Redundant API calls
- Context bloat opportunities

Provide:
1. Current monthly cost
2. Optimized monthly cost
3. Specific changes to implement
4. Estimated savings %

Output as: actionable_report.md
```

---

## 3️⃣ EMAIL SEQUENCE BUILDER

```
Create a 5-email nurture sequence for [PRODUCT/SERVICE].

Audience: [Description]
Goal: Convert to [ACTION]

Structure each email:
- Subject line (curiosity-driven)
- Preview text (15 words)
- Body (150-200 words)
- CTA (single, clear)

Email cadence:
1. Day 0: Welcome + value
2. Day 2: Problem awareness
3. Day 4: Solution intro
4. Day 6: Social proof
5. Day 8: Urgency + offer

Tone: Conversational, not salesy
```

---

## 4️⃣ COMMENT RESPONDER

```
You are monitoring comments on my [PLATFORM] content.

For each comment, respond with:
- Helpful answer (if question)
- Appreciation (if positive)
- Thoughtful counter (if negative)
- CTA to Discord (if interested)

Rules:
- Max 2 sentences
- Natural tone (not robotic)
- Include emoji (1-2 max)
- Never be defensive

If comment is spam/hate → ignore (don't respond)
```

## 5️⃣ COMPETITOR ANALYSIS

```
Research [COMPETITOR] and analyze their strategy.

Investigate:
- Content topics (top 10 videos/posts)
- Posting frequency
- Engagement rates
- Monetization methods
- Unique angles

Provide:
1. What they do well (3 things)
2. Gaps in their content (3 opportunities)
3. Differentiation strategy for me
4. Action plan (5 steps)

Output: competitor_analysis_[NAME].md
```

## 6️⃣ TREND IDENTIFIER

```
Find trending topics in [NICHE] right now.

Sources:
- Twitter trending hashtags
- Reddit hot posts (r/[subreddit])
- YouTube trending (past 7 days)
- Google Trends (rising queries)

For each trend, provide:
- Topic summary
- Why it's trending
- Content angle for me
- Urgency level (hot/warm/cold)

Output top 5 trends ranked by opportunity.
```

## 7️⃣ SALES PAGE OPTIMIZER

```
Review this sales page copy and optimize for conversion.

Page: [URL or paste text]
Goal: [Increase opt-ins / sales / signups]

Analyze:
- Headline clarity
- Value proposition strength
- Objection handling
- CTA placement
- Trust signals

Provide:
1. Current conversion blockers (3-5)
2. Rewritten sections (before/after)
3. A/B test ideas (3 variants)
4. Expected impact estimate

Format: side-by-side comparison
```

Design an automation workflow for: [GOAL]

Requirements:
- Start trigger: [EVENT]

- Constraints: [BUDGET/TIME/TOOLS]

Map out:
1. Trigger event

## 9️⃣ LINKEDIN POST GENERATOR

```
Create 5 LinkedIn posts about [TOPIC].

Format for each:
- Hook (first line - must stop scroll)
- Story or insight (3-5 lines)
- Takeaway (actionable)
- CTA (comment/DM/link)

Post types:
1. Personal story
2. Controversial take
3. How-to guide
4. Industry observation
5. Wins/results

Tone: Professional but personal
Length: 100-150 words each
```

## 🔟 PRODUCTIVITY AUDIT

```
Analyze how I spent my time this week.

Review:
- Calendar events
- Task completion
- Time blocks vs actual time
- Distractions/interruptions

Categorize time:
- High-value work (revenue-generating)
- Medium-value (important but not urgent)
- Low-value (could delegate/automate)
- Waste (eliminate)

Provide:
1. Time breakdown (% per category)
2. Automation opportunities (5 tasks)
3. Delegation candidates (3 tasks)
4. Schedule optimization (new template)

Goal: Free up 10 hours/week
```

## 🎯 HOW TO USE THESE

### Method 1: Direct Command
```bash
openclaw run "prompt_text_here"
```

### Method 2: Save as Task File
Create `tasks/content-generator.md`:
```markdown
# Content Generator
[Paste prompt #1 here with your specific details]
```

Then run:
```bash
openclaw run tasks/content-generator.md
```

### Method 3: Scheduled Automation
Add to crontab:
```bash
# Generate daily content at 9 AM
0 9 * * * openclaw run tasks/daily-content.md
```
