## Technical Notes
- Shopify CDN cache is aggressive - changes take 5-10 minutes to propagate
- Dawn theme uses `!= blank` checks in Liquid - empty string "" is not blank, uses schema defaults
- To hide default button text, either modify liquid or set explicit non-blank values
- `show_text_below` in image-banner only adds `banner--mobile-bottom` class (mobile only)
