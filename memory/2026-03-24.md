## CLEVERDOGMETHOD AUTOMATION SYSTEM — DEPLOYED ✅

### **Project Overview**
Built complete automation system for cleverdogmethod.com to generate 3 unique, SEO-optimized dog training blog posts daily at 9:00 AM Spain time (8:00 UTC).

### **System Architecture**

**4 Core Components:**

1. **Research Pipeline** (`scripts/research/`)
   - `reddit_scraper.py` — Scrapes r/Dogtraining, r/dogs, r/puppy101 for trending topics
   - `google_trends.py` — Fetches rising dog training search queries (with fallback)
   
2. **Content Generator** (`scripts/content/`)
   - `content_generator.py` — AI-powered blog writer using Claude 3.5 Sonnet
   - Generates 800-1200 word posts with SEO optimization
   - Personal tone, 8th grade readability, natural affiliate CTA

3. **Quality Control** (`scripts/deploy/`)
   - `uniqueness_checker.py` — Prevents duplicates via title/keyword similarity
   - Rejects posts with >70% title match or >60% keyword overlap

4. **Deployment** (`scripts/deploy/`)
   - `deploy_posts.py` — Saves HTML, updates sitemap, git push, Telegram notify
   - Auto-deploys to Netlify (live in ~2 min)

**Main Orchestrator:**
- `cleverdogmethod-daily.py` — Coordinates all 4 components in pipeline
